# WeChatGPT
微信公众号版ChatGPT，私信即可与ChatGPT交谈

.net 6版本的代码
MySQL数据库
对接OpenAI官方接口（需要科学上网）

在公众号后台设置好回调地址（该项目中/api/wechat）以及token

![image](https://user-images.githubusercontent.com/19923325/224868344-0561d791-46ed-4010-a4be-2800efff7d11.png)

在项目中配置openai的key，连接字符串即可使用
